## UART_LED_Blink
Bread Board 에 LED 3개를 연결하고, PC에서 입력한 숫자만큼 LED1, LED2, LED3를 깜빡입니다.
ex) 숫자 4를 입력하면 LED 전체가 4번 깜빡임
